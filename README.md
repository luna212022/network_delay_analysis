# Network Delay Analysis using Mininet

This project analyzes network delay in different topologies using Mininet.

## Files included:
- delay_measure.py → Python script to measure delay
- Screenshots → Output of Mininet experiments

## Experiments:
- Linear topology connectivity
- Ping delay measurement
- Larger topology delay comparison
- Flow table entries in switch

## Tools used:
- Mininet
- Python
- Open vSwitch
